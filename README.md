## Local Development Setup (VS Code)

### Prerequisites

- [Node.js & npm](https://github.com/nvm-sh/nvm#installing-and-updating) (v18 or later)
- [Supabase CLI](https://supabase.com/docs/guides/cli/getting-started) (for local edge functions)
- VS Code (recommended extensions: ESLint, Prettier, Tailwind CSS IntelliSense)

### Setup Instructions

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>

# Step 2: Install dependencies
npm install

# Step 3: Set up environment variables
# The .env file is auto-generated by Lovable Cloud
# It contains:
# - VITE_SUPABASE_URL
# - VITE_SUPABASE_PUBLISHABLE_KEY
# - VITE_SUPABASE_PROJECT_ID

# Step 4: Start the development server
npm run dev
```

The app will be available at `http://localhost:8080`

### Testing Edge Functions Locally

The project uses Supabase Edge Functions (Deno runtime) for AI recipe generation.

```sh
# Install Supabase CLI (if not already installed)
# macOS/Linux
brew install supabase/tap/supabase

# Windows
scoop bucket add supabase https://github.com/supabase/scoop-bucket.git
scoop install supabase

# Link to your Supabase project
supabase link --project-ref ynuhnjamnirbjbdusjeq

# Set up local secrets for edge functions
# Create a .env file in supabase/.env with:
# LOVABLE_API_KEY=your_lovable_api_key

# Start Supabase services locally
supabase start

# Serve edge functions locally
supabase functions serve --env-file supabase/.env

# Test the generate-recipe function
curl -i --location --request POST 'http://localhost:54321/functions/v1/generate-recipe' \
  --header 'Content-Type: application/json' \
  --data '{
    "ingredients": "rice, lentils, turmeric",
    "cuisine": "south",
    "preference": "veg"
  }'
```
